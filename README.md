# CellularJS HTTP Sample

## 1. CRUD
_There is no DB migration available now. You can only see code._

```bash
$ git clone https://github.com/cellularjs/http-sample.git
$ cd http-sample
$ yarn
$ yarn dev:crud
```

## 2. Worker
